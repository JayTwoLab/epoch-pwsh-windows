# epoch-pwsh-windows

> [Korean](README.md)

- A simple script to easily view `epoch` time as a command on Windows
  - `epoch` is the number of seconds since January 1, 1970 (UTC).
  - `1` : 1970-01-01 00:00:01

## Installation

- Copy 'epoch_time.ps1' and 'epoch.cmd' into the 'PATH' path. 

## Usage

- Run the `epoch` command from Windows Terminal or PowerShell, for example:
```bat
PS C:\UTIL> epoch 1718712000
1718712000
--------------------------------
Epoch Time : 1718712000
UTC Time   : 2024-06-18 12:00:00
Local Time : 2024-06-18 21:00:00
--------------------------------
```
